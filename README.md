# Pandas Data Analysis 🐼

## Description
This repository contains a collection of Python data manipulation and analysis projects built entirely in **Google Colab**. Using the **Pandas** library, this project focuses on importing, cleaning, and transforming datasets to extract meaningful data insights.

Because this project is built for Google Colab, it requires zero local software setup and runs completely in the cloud.

## Features
*   **Data Cleaning:** Handling missing data, dropping duplicates, and correcting data types.
*   **Data Transformation:** Sorting, filtering, merging, and aggregating complex datasets.
*   **Exploratory Data Analysis (EDA):** Grouping data and generating descriptive statistics.

## How to Run This Project
You don't need to install Python or set up a virtual environment on your computer. You can run these notebooks directly in your browser:

1. Click on the `.ipynb` notebook file inside the `notebooks/` folder above that you want to view.
2. Replace the URL in your browser or click the badge inside the notebook to open it in Google Colab.
3. To execute the code, select **Runtime > Run all** from the top menu in Colab.

## Project Structure
*   `notebooks/` — Google Colab notebooks (`.ipynb` files) containing the data analysis steps.
*   `data/` — Sample datasets (CSV, Excel, or JSON) used in the analysis.
